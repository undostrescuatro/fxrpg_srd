This is a custom RPG made by 「プチ」Haku.
FxRPG is a Free Open Source Role Playing Game.

The name Fx stands for 'Effect' this being one of the main guiding principles in the design. The rules are centered around Degrees of Effect, and their consequences. If something does not produce an effect on the way a character is played, it tends to be ignored by the rules.

This is done in order to minimize abstract numbers like hit points, though sometimes this is unavoidable, the goal is to keep this to a minimum.
