# Equipment Situations

## Equipment Modifiers

Some Skills rely heavily on tools or equipment, the following table shows how the quality of tools affect both Test and Work efficiency:

| Equipment  | Test Modifier | Work Modifier |
| ---------- | ------------- | ------------- |
| Absent     | -2            | 1/4           |
| Improvised | -1            | 1/2           |
| Basic      | 0             | 0             |
| Good       | +1            | 1.2           |

## Repair

Repairs are handled using the appropriate Skill.

- Mechanical Skill for mechanical systems
- Electronics Skill for Electrical system
- Or Lore, Trade, or Craft Skills for relevant scenarios.

Repairs are treated as Work Tests. Spare parts are included in the repair cost, but tools and their quality depend on the Equipment Modifier. Repair Thresholds are based on the highest Failure gained by the machine.

| Damage   | Column B |
| -------- | -------- |
| Light    | 2        |
| Heavy    | 4        |
| Critical | 6        |

| Damage                      | Work Points       | Cost |

| Fire Control                | $2*Rating$        |      |
| ACC                         | 1 per rating lost |      |
| AUX Equipment Destroyed     | $5$               |      |
| Crew Compartment Failure    | $10$              |      |
| Complete Structural Failure | $Size^2$          |      |
| SpecialFx                   | $Cost$            |      |

Technicians can contribute to the task individually or as a Group Test, combining their WP contributions until the total is met. Fumbles on the Work Test produce a Light damage result.
