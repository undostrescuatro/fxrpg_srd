# Economic Situations

## Availability

Use these rules when players are trying to purchase uncommon or restricted items.

Restricted items have an availability code written as A-CV, interpreted as:

- A: Availability Threshold. The higher the number, the harder it is to find.
- C: Restriction code. Who can legally own it.
- V: Violation level. How severe the penalty is for Illegal possession.

### Finding an Object

- Roll 2d6, vs the item availability Threshold.
- You can add +1 Dice for every 10% price increase.

Result:

- Success: The item is found.
- Failure: The item is not found.
- Fumble: A relevant authority is alerted.
- Tie: The item is found.

At the Referee's discretion, A Skill Test may replace the basic 2d6 roll (e.g., Negotiation, Etiquette)

### Time to Find the Object

Use the Degree to determine how long it the search takes.

| Degree | Time Frame               |
| ------ | ------------------------ |
| -3     | Full day (4d6 Hours)     |
| -2     | Work Day (2d6 Hours)     |
| -1     | Half Workday (1d6 Hours) |
| 0      | Full day (4d6 Hours)     |
| +1     | Work Day (2d6 Hours)     |
| +2     | Half Workday (1d6 Hours) |
| +3     | Hour (1d6\*10 Minutes)   |

| Availability Threshold | Description                             |
| ---------------------- | --------------------------------------- |
| 1                      | Very common; anytime, anywhere          |
| 2-4                    | Common; cam be found with little effort |
| 5                      | Uncommon; may require some work to find |
| 6                      | Hard to find                            |
| 7                      | Very hard to find                       |
| 8                      | Restricted objects                      |
| 9                      | Limited production runs                 |
| 10                     | Only a few units in existence           |
| 11                     | Prototypes                              |
| 12                     | One of a kind                           |

| Restriction | Class      | Description                                                                                                         |
| ----------- | ---------- | ------------------------------------------------------------------------------------------------------------------- |
| I           | Illegal    | Transportation, possession, use, sale, or storage is illegal.                                                       |
| M           | Military   | Transportation, possession, use, sale, or storage restricted to only military.                                      |
| P           | Police     | Transportation, possession, use, sale, or storage restricted to only law enforcement.                               |
| R           | Registered | Transportation, possession, use, sale, or storage restricted to civilians with a Registered documentation (permit). |
| E           | Everyone   | Unrestricted                                                                                                        |

| Violation | Description                                                                                                                                                     |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0         | No Punishment                                                                                                                                                   |
| 1         | 1 Month time-frame (1d6 weeks) + community service; therapy with personal psychiatrists; repayment of damage; Half a year time-frame (1d6+1 months) probation   |
| 2         | Half a year time-frame (1d6+1 months) + Minor Fine; community service; extensive therapy with psychiatrist; any combination of the above; (1d6 years) probation |
| 3         | Half a year time-frame (1d6 Months) +Large fine; incarceration in rehabilitation center; charge entered on permanent record                                     |
| 4         | 1d6 Years + Huge fine; incarceration in rehabilitation center; charge entered on permanent record                                                               |
| 5         | 3d6 Years + Huge fine; incarceration; permanent record (75% of parole after half the term has been served)                                                      |
| 6         | 5+3d6 Years + Huge fine; incarceration; permanent record (50% of parole after half the term has been served)                                                    |
| 7         | Life sentence (25% to 50% chance of parole after 10+3d6 years)                                                                                                  |
| 8         | Life sentence; no parole or death penalty (where applicable)                                                                                                    |
